Para la instalación de react:

1- Crear una carpeta cualquiera
2- Abrir la carpeta y en su ruta borrar y escribir "cmd".
3- Escribir el siguiente comando

npm init vite@latest nombredemiaplicacion -- -- template react
latest: es la última versión
nombre de la aplicación....
Nota: deben respetarse los espacios entre los guiones o no funcionará. Presiono enter y a esperar... si pregunta algo probablemente para seguir deba colocar Y. Una vez q se instalo todo, colocar code . (espacio punto), y con eso se abre la carpeta en el visual studio code.
